# @type-config/easy

The most loosing rules for typescript. Fit for new starter or old javascript projects.

## rules
```json
{
  "compilerOptions": {
    "module": "commonjs",
    "target": "es6",
    "sourceMap": true,

    "moduleResolution": "node",
    "esModuleInterop": true,
    "allowJs": true,
    "allowUnreachableCode": true,
    "allowUnusedLabels": true,
    "noStrictGenericChecks": true,
    "skipLibCheck": true
  }
}
```
